## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

### Setup for Running Locally on Port 3000

From within the root directory:

```sh
npm run build:dev
```

In a separate terminal, run the server:

```sh
npm run start
```
